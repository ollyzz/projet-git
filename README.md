# TD 3 : Dépôt distant, partage de code et travail collaboratif

Ce dépôt contient les travaux réalisés durant le **TD 3**, portant sur la gestion des dépôts distants avec GitHub et les flux de travail collaboratifs (Push, Clone, Pull, Fetch et Merge).

## Objectifs du TD
L'objectif était d'apprendre à partager son code, à configurer une sécurité via SSH et à collaborer avec un partenaire sur un même projet.
